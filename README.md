# 💰 SmartSpend — AI Expense Tracker

SmartSpend is an intelligent expense tracking web application that helps users monitor, analyze, and optimize their spending using AI-powered insights.

---

## 🚀 Features

### 🧠 Auto-Categorization

* Simply type an expense description
* Category is automatically detected using smart logic
* Example:

  * “Starbucks” → Food
  * “Uber” → Transport

---

### 📊 Monthly Insights

* Visual breakdown of spending using charts
* Smart observations about your spending habits
* Identify where most of your money goes

---

### ⚠️ Overspending Alerts

* Get warnings when you're:

  * Approaching budget limits
  * Exceeding category budgets
* Helps maintain financial discipline

---

### 📈 AI Predictions

* Predicts your end-of-month expenses
* Based on your current daily spending rate
* Gives you a forward-looking financial view

---

### ➕ Expense Management

* Add new expenses via a sleek modal UI
* Delete expenses easily
* Clean and intuitive user experience

---

## 🧠 Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** Tailwind CSS
* **State Management:** React Hooks
* **Charts:** Recharts
* **UI Components:** shadcn/ui
